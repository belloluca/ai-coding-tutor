# AI Coding Tutor

AI Coding Tutor is a web application developed as a Bachelor's Degree thesis project in Computer Science. Its goal is to support students in learning programming through the use of Artificial Intelligence.

The application provides a simple and intuitive interface that allows users to interact with an AI-powered coding assistant, offering support through multiple learning modes.

## 🚀 Features

* 📖 **Tutor** – Explains theoretical and practical programming concepts.
* 🐞 **Debug** – Analyzes source code, identifies errors, and suggests possible fixes.
* 💻 **Exercises** – Generates personalized programming exercises for practice.
* ❓ **Quiz** – Creates quizzes to assess and reinforce learning.
* 👤 User registration and login system.
* 📧 Email verification during registration.
* 💬 Automatic conversation saving.
* 📚 Chat history available for each user.

## 🛠️ Technologies Used

### Front-end

* HTML5
* CSS3
* JavaScript

### Back-end

* Python
* Flask
* Flask-CORS

### Database

* SQLite

### AI

* Google Gemini API

## 🎯 Project Goal

The purpose of this project is to demonstrate how Large Language Models (LLMs) can be integrated into a web application to provide personalized support for learning programming through explanations, exercises, quizzes, and debugging assistance.
