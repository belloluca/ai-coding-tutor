# AI Coding Tutor

AI Coding Tutor è un'applicazione web sviluppata come progetto di tesi triennale in Informatica con l'obiettivo di supportare gli studenti nello studio della programmazione attraverso l'utilizzo dell'Intelligenza Artificiale.

L'applicazione mette a disposizione un'interfaccia semplice e intuitiva che permette di interagire con un assistente AI specializzato nel coding, offrendo supporto in diverse modalità.

## 🚀 Funzionalità

- 📖 **Tutor** – Spiegazione di concetti teorici e pratici di programmazione.
- 🐞 **Debug** – Analisi del codice e individuazione degli errori con suggerimenti per la correzione.
- 💻 **Esercizi** – Generazione di esercizi personalizzati per esercitarsi.
- ❓ **Quiz** – Domande per verificare il livello di apprendimento.
- 👤 Sistema di registrazione e login degli utenti.
- 📧 Conferma dell'email durante la registrazione.
- 💬 Salvataggio automatico delle conversazioni.
- 📚 Storico delle chat consultabile dall'utente.

## 🛠️ Tecnologie utilizzate

### Front-end
- HTML5
- CSS3
- JavaScript

### Back-end
- Python
- Flask
- Flask-CORS

### Database
- SQLite

### AI
- Google Gemini API

## 🎯 Obiettivo

Il progetto nasce con lo scopo di dimostrare come i Large Language Models possano essere integrati in un'applicazione web per fornire un supporto personalizzato allo studio della programmazione, migliorando l'apprendimento attraverso spiegazioni, esercizi e assistenza al debugging.
