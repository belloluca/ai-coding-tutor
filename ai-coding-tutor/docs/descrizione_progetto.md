# Descrizione del progetto

Il progetto consiste nella realizzazione di una web app chiamata **AI Coding Tutor**, progettata per supportare gli studenti nello studio della programmazione.

L'applicazione è composta da due parti principali:

1. **Front-end**, cioè l'interfaccia grafica con cui interagisce lo studente.
2. **Back-end**, cioè la parte logica che riceve le richieste, genera le risposte e salva le interazioni.

## Funzionamento generale

Lo studente scrive una domanda o incolla del codice nell'interfaccia della chat. Il front-end invia il messaggio al back-end tramite una richiesta HTTP. Il back-end elabora il messaggio in base alla modalità selezionata e restituisce una risposta.

Le modalità previste sono:

- Tutor
- Debug codice
- Esercizi
- Quiz

## Utilità nella tesi

Il progetto può essere utilizzato come parte sperimentale della tesi, mostrando un esempio pratico di utilizzo dell'AI nell'istruzione e nello studio del coding.

La cartella `sperimentazione` può contenere questionari, risultati dei test e grafici ottenuti dopo aver fatto provare il sistema ad alcuni utenti.
