# Sperimentazione

Questa cartella può contenere il materiale usato per valutare il progetto.

Esempi:

- questionari compilati dagli utenti;
- risultati dei test;
- grafici;
- tabelle di valutazione;
- osservazioni raccolte durante l'utilizzo del chatbot.

Possibili criteri di valutazione:

- chiarezza delle risposte;
- utilità nello studio;
- correttezza tecnica;
- semplicità dell'interfaccia;
- soddisfazione generale dell'utente.
